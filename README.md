# Numan and Associates ERP System

This repository contains a Vite React ERP dashboard built for a client by **Numan and Associates**.

## Project Overview

- Built with React, TypeScript, and Vite.
- Uses Tailwind-style utility classes for responsive UI.
- Integrates with Supabase for persistent data storage.
- Includes modules for:
  - Home dashboard
  - Taxation services
  - Amazon services
  - Law services
  - Immigration services
  - Language services
  - Investment services
  - Academic services
  - Marketing services
  - Training services
  - AI assistant
  - Calendar & reminders
  - Reports & analytics
  - Search & filters
  - Settings

## Deployment

- Ready for deployment on Vercel with `vercel.json` configured.
- Build command: `npm run build`
- Output directory: `dist`

## Environment Variables

Create a `.env` file with the following values:

```env
VITE_SUPABASE_URL=your_supabase_url
VITE_SUPABASE_ANON_KEY=your_supabase_anon_key
```

## Notes

- Local storage is used only for profile settings and saved search filters.
- Most application data is persisted to Supabase.
- The project is mobile-friendly and includes a mobile nav menu.
